# web-assignment-shaked-and-roni

This is a server-side application that exposes a RESTful API for managing **Posts** and **Comments**.  
The API supports full CRUD operations (Create, Read, Update, Delete).

---

## Features

- Posts
  - Create a new post
  - Get all posts
  - Get a single post by ID
  - Get all posts written by a specific sender
  - Update an existing post
- Comments
  - Add comments to a post
  - Get comments by post
  - Get all comments
  - Get a single comment by ID
  - Update a comment
  - Delete a comment

---

### Installation & Run

# Install dependencies
npm install

# Start the server
npm run start
